Here's my story

I love to <code>. Ever since my first adolescent web design class back in high school, I was hooked. I've built 3 PC's so far, and nearly all the money I've earned working part-time jobs has one way or another been found to either add more RAM, install an all-in-one water cooler, or upgrade my ancient hard drive to a sleek SSD.
